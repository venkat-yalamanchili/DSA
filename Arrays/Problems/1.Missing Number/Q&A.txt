Missing Number
Write a function to find the missing number in a given integer array of 1 to 100. 
The function takes to parameter the array and the number of elements that needs to be in array.  
For example if we want to find missing number from 1 to 6 the second parameter will be 6.

Example : missing_number([1, 2, 3, 4, 6], 6)
Output  : 5


Solution: You can use the arithmetic progression formula to find the sum of the numbers from 1 to n and then subtract the sum of the numbers in the array to get the missing number
